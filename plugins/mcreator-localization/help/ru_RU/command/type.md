Эта опция указывает тип данной команды и допустимую область её использования.

* **Стандартная**: Команда доступна без ограничений. Стандартное поведение команд (например, `/give`).
* **Однопользовательская**: Команда доступна только в одиночном режиме (например, `/publish`).
* **Многопользовательская**: Команда доступна только в многопользовательском режиме (например, `/ban`).
* **Клиентская**: Команда доступна только на стороне клиента, вызываемые ей действия не могут затронуть сторону сервера.
