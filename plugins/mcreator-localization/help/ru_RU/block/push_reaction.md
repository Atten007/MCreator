Этот параметр задаёт реакцию данного блока на поршень.

* **По умолчанию:** Стандартное взаимодействие с поршнями (подобно камню);
* **Разрушение:** При попытке перемещения поршнем блок будет разрушен (подобно яйцу дракона);
* **Блокировка:** Блок не сдвигается поршнем (подобно обсидиану);
* **Только сдвиг:** Поршень сможет только толкать этот блок.
