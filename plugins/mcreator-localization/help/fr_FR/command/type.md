Cette option vous permet de déterminer le type de la commande et où/comment elle peut être utilisée.

* **Standard** : La commande peut être utilisée partout. Il s'agit du comportement normal (par exemple `/give`).
* **Uniquement en solo** : La commande sera disponible uniquement dans les mondes solo (par exemple `/publish`).
* **Uniquement en multijoueur** : La commande sera disponible uniquement sur les serveurs multijoueur (par exemple `/ban`).
* **Côté client** : La commande sera exécutée exclusivement côté client, ce qui signifie que seules des actions sur le client peuvent être effectuées.
